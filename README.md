# Tranj Next (WIP)

Next.js website for a meat production company

Emphasis on server rendering and SEO optimization

## Getting Started

```bash
pnpm install
pnpm dev
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in the values:

```bash
cp .env.example .env.local
```

## Tech Stack

- Next.js
- next-intl
- HeroUI
- Strapi (backend)
